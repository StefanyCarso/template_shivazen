# Template Shivazen

## Sobre o Projeto

Este repositório contém um template front-end desenvolvido para integração com o projeto principal Shivazen. O template foi construído utilizando apenas tecnologias front-end simples (HTML, CSS e JavaScript), com o objetivo de fornecer uma interface leve, responsiva e de fácil manutenção para o sistema principal.

O template serve como camada de apresentação do ecossistema Shivazen, permitindo uma separação clara entre a interface do usuário e a lógica de negócio implementada no back-end.

## Template Base

Este projeto é derivado e adaptado do seguinte template:

- **Template Name:** Clinic
